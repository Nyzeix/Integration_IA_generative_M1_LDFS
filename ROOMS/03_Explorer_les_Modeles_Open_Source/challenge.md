# Challenge — Room 03

## Objectif

Comparer les trois modèles sur une tâche plus complexe : le résumé de texte.

## Défi

1. Prenez le fichier `datasets/texte_entreprise.txt` fourni dans le dépôt.
2. Envoyez-le aux 3 modèles avec le prompt : "Résume ce texte en 5 phrases claires et concises."
3. Comparez les résumés obtenus dans un tableau enrichi avec les critères suivants :
   - **Fidélité** : le résumé reflète-t-il correctement le contenu original ?
   - **Concision** : le résumé respecte-t-il la contrainte de 5 phrases ?
   - **Lisibilité** : le résumé est-il agréable à lire ?
   - **Informations manquantes** : des éléments importants ont-ils été omis ?

## Livrable

Un fichier `challenge_room03.md` dans `expected_outputs/` contenant :
- Les 3 résumés produits
- Le tableau enrichi rempli
- Un paragraphe de conclusion : quel modèle est le plus adapté au résumé et pourquoi ?
