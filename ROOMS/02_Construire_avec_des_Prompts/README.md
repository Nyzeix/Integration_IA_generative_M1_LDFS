# Room 02 — Construire avec des prompts

## Objectif

Apprendre à construire des prompts efficaces pour obtenir des réponses précises, fiables et exploitables.
A la fin de cette Room, vous serez capable de transformer un prompt vague en prompt structuré, d'obtenir une sortie en JSON et de construire un assistant pédagogique simple.

## Résultat attendu

- Un prompt vague transformé en prompt structuré, avec comparaison des deux sorties
- Une réponse formatée en JSON valide, exploitable par un programme Python
- Un assistant pédagogique fonctionnel en ligne de commande
