# Room 06 — Comprendre les risques

## Objectif

Identifier et analyser les principaux risques liés à l'utilisation de l'IA générative : hallucinations, biais, données personnelles, risques juridiques et prompt injection.

## Résultat attendu

- Une grille de vérification de faits remplie pour 5 réponses de LLM
- Une analyse documentée de biais détectés dans les réponses
- Une liste de 5 bonnes pratiques pour un usage professionnel
- Un protocole d'audit de réponses générées
