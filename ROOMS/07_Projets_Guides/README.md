# Room 07 - Projets guidés

## Objectif

Mettre en pratique toutes les compétences des Rooms précédentes dans 3 projets encadrés. Chaque projet combine plusieurs notions et produit un assistant fonctionnel.

## Les 3 projets

| Projet | Titre | Compétences mobilisées |
|--------|-------|----------------------|
| A | Assistant mémoire | Prompt avec historique, gestion du contexte |
| B | Assistant entreprise | RAG, citation des sources, gestion de documents |
| C | Assistant analyse de texte | Prompt structuré, sortie JSON, traitement par lots |

## Structure de chaque projet

Chaque projet contient dans `code/projet_X/` :
- `architecture.md` - description de l'architecture du projet
- Un script Python principal à compléter
- Des fichiers de test

## Fichiers de cette Room

| Fichier | Contenu |
|---------|---------|
| `theory.md` | Récapitulatif des notions nécessaires pour les 3 projets |
| `practice.md` | Consignes détaillées pour chaque projet |
| `challenge.md` | Extensions possibles pour chaque projet |
| `rubric.md` | Critères d'évaluation |
