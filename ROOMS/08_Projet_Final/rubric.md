# Rubric - Room 08 : Projet final

## Barème (sur 40 points)

| Critère | Points | Ce qui est évalué |
|---------|--------|-------------------|
| Architecture | 6 | Schéma clair, composants identifiés, flux de données décrit |
| Code fonctionnel | 10 | Scripts exécutables sans erreur, commentés, modulaires |
| Intégration RAG | 6 | Base documentaire indexée, recherche pertinente, réponse contextualisée |
| Prompt engineering | 4 | Prompts structurés, rôle défini, contraintes de sortie |
| Analyse des risques | 6 | Au moins 3 risques identifiés, mesures de mitigation proposées |
| Rapport | 6 | Complet, structuré, choix justifiés, limites documentées |
| Démonstration | 2 | Session réaliste, cas nominal et cas limite montrés |

## Bonus

| Bonus | Points |
|-------|--------|
| Challenge extension fonctionnelle | +4 |
| Qualité exceptionnelle du code et de la documentation | +2 |

## Critères éliminatoires

- Le projet ne s'installe pas avec `pip install -r requirements.txt` → note maximale 20/40
- Aucun composant RAG → note maximale 25/40
- Aucune analyse des risques → note maximale 25/40
